G7 SBOM for AI - Minimum Elements

Namespace: https://w3id.org/sbom/req/g7ai/  |  g7ai.ttl |  v0.1.0

Concepts derived from 'Software Bill of Materials (SBOM) for Artificial Intelligence - Minimum Elements', published by the G7 Cybersecurity Working Group on 2026-05-12. Categories: Metadata (MD), System Level Properties (SLP), Models (MDL), Dataset Properties (DP), Infrastructure (INF), Security Properties (SP), Key Performance Indicators (KPI).
